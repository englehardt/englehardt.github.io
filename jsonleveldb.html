JSONLevelDB

Date Tue 02 September 2014 Tags Code

LevelDB is a simple and fast key-value store. It supports a simple API to Push, Get, and Delete (key, value) byte-string pairs to and from the database with a few extra options for batched writes. Similar to SQLite, it's an excellent embedded storage option but should provide better performance.

py-leveldb wraps the C++ library and replicates most of the API. This includes the requirement that both the key and value be byte-strings (str in 2.x and byte in 3.x). JSONLevelDB wraps the API exposed by py-leveldb to include an automatic serialization to JSON when using Put (both directly and for WriteBatch objects) and de-serialization when using Get.
